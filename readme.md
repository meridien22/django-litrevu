# # LITRevu : le site des passionnées de littérature

L’application est dédiée à la littérature et permet de :
- demander des critiques de livres ou d’articles, en créant un billet ;
- lire des critiques de livres ou d’articles ;
- publier des critiques de livres ou d’articles.


Etapes à suivre pour installer localement l'application :

1. **Cloner le repository GitHuh du projet**
    * `git clone https://github.com/meridien22/django-litrevu.git`
2. **Aller dans le répsertoire du projet**
    * `cd litrevu/`
3. **Créer l'environnement virtuel**
    * `python -m venv venv`
4. **Activer l'environnement virtuel**
    * `source venv/bin/activate`
5. **Installer les paquets nécessaires**
    * `pip install -r requirements.txt`
6. **Aller dans le répertoire de l'application**
    * `cd litrevu/`
6. **Lancer le serveur Django**
    * `python manage.py runserver`
7. **Dans votre navigateur, rendez-vous à cette adresse**
    * `http://127.0.0.1:8000/`
